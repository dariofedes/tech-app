# geimmeLists! App

## Introduction

This app shows a placeholder list of 100 items. It requires authentication to show it.

## Install dependencies: 

```shell
$ npm i
````

## Run app



Run app in iOS emulator: 

```shell
$ npm run ios
````

Run app in android emulator: 

```shell
$ npm run android
````

## Testing

Unit testing with data integration using Jest.

Run tests:

```shell
$ npm test
````
